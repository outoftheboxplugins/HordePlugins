# Horde Plugins

First-party plugins for the [Horde](https://dev.epicgames.com/documentation/unreal-engine/horde) build system, published as `@outoftheboxplugins/horde-*` (npm) and `OutOfTheBoxPlugins.Horde*` (NuGet).

## Plugins

| Plugin | Description |
|--------|-------------|
| [Storage Reporter](StorageReporter/) | Breaks down artifact distribution across CI builds |
| [Tools Uploader](ToolsUploader/) | Drag-and-drop tool deployment upload via the dashboard |
| [Discord Notifications](DiscordNotifications/) | Build notifications via Discord webhooks |
| [Email Notifications](EmailNotifications/) | Build notifications via SMTP email |
